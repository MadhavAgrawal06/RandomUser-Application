# Random Users React App

This project was created as part of the **Tericsoft Frontend Assessment**.

It is a small React application that fetches random user data from a public API and displays it in a clean and responsive UI.

## Features
- Fetches data from a public API
- Displays user photo, name, and country
- Handles loading and error states
- Responsive layout

## API Used
https://randomuser.me/api/

## Tech Stack
- React
- JavaScript
- Vite
- CSS Grid

## How to Run

1. Clone the repository

```bash
git clone https://github.com/MadhavAgrawal06/RandomUser-Application.git
```

2. Navigate to the project folder

```bash
cd react-api-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

## Output

The app fetches random users and displays:
- Profile photo
- Name
- Country
